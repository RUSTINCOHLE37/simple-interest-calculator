# Contributing to Simple Interest Calculator

Welcome! We appreciate your interest in contributing to our project. Here are some ways you can help:

- **Bug Reports**: If you find a bug, please let us know so we can fix it.
- **Bug Fixes**: If you have a solution for an existing bug, we welcome your pull requests.
- **Documentation Improvements**: Help us enhance our documentation to make it clear and useful for everyone.
- **Enhancements**: Any suggestions for new features or enhancements are greatly appreciated.
- **Ideas**: We're open to new ideas that can help improve the project.

Thank you for considering contributing!
